Diff matrix

The so called difference matrix shows rank matrix data in a slightly different way, by subtracting a player's deaths from his frags. This makes it very clear who is the victor and who is the victim.

The matrix used to calculate the stats on the dm rank page is x in size. This page only shows a top left subset ( x ) of the rank matrix, that is incident sorted.

Player list

This is the subset list of of all the players in ranking, the folks with the most incidents ( of the total ). Names are alphabetically sorted. See Help for details.

Rank matrix

This is a subset of the full ranking matrix, since it is incident sorted, the folks that played the most are found in the top left. See Help for details.

Intensity rank matrix

This matrix shows the same data as the one above, only now the data is presented as an intensity range. This makes it easier to pick out high (red) and low (blue) values. See Help for details.
